📋 How auto-allocation works

Banksia OS will automatically match each document to the right tenancy or tenant by scanning the filename for:

• Tenancy references — e.g. TE1578_agreement.pdf, Tenancy_918_AST.pdf
• Tenant names — e.g. Safoora_Kumbalakkuzhi_referencing.pdf
• Property addresses — e.g. 32_Rectory_Square_EICR.pdf
• Arthur-style exports — any filename pattern

Unmatched files will be flagged for manual assignment from the dashboard.
